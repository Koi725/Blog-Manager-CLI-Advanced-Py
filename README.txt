# 📝 CLI Blog Manager (Advanced Python Project)

A clean, modular, and feature-rich command-line blog/post management system written in Python.  
This project handles user authentication, post creation, likes, comments, and more — fully offline and JSON-based.  
Part of my **30 Project Series** (Advanced Level) on the path to becoming a Senior Backend Developer.

---

## 🚀 Features

- 🔐 **User Authentication** (Sign-up / Login with validation)
- 📝 **Create/Edit/Delete Posts**
- 💬 **Comment System** (with username and timestamp)
- ❤️ **Like System** (each user can like once)
- 🔍 **Search Posts** (by title or content)
- 👤 **View Posts by User**
- 📚 **View All Posts** and **Post Details**
- 🧠 Clean code architecture using `utils/` module system
- 📦 Data persisted in JSON files (`user.json`, `posts.json`)
